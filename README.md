# Pixel Studio

Een pixel art editor gemaakt met HTML, CSS en JavaScript.

## Bestanden

- index.html — hoofdpagina
- style.css — styling
- main.js — logica en classes

## Starten

Open index.html in je browser.

## Classes

- App — start de applicatie
- Grid — beheert het raster
- Pixel — één cel in het raster
- PenTool — tekenen
- EraserTool — gummen

## Functies

- Pixels kleuren via klikken
- Kleur kiezen met ColorPicker
- Gummen en alles wissen
